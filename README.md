# SmartLens - Real-time Object Detection

Android app using TensorFlow Lite for on-device object detection with live camera feed.

## Features
- Real-time object detection using MobileNetV2
- 1000+ object classifications
- Photo capture with labeled results
- Material Design 3 UI
- Background inference with Kotlin Coroutines

## Tech Stack
- Kotlin
- Jetpack Compose
- TensorFlow Lite
- CameraX
- ViewModel & State
- Material 3

## Setup
1. Clone repository
2. Open in Android Studio
3. Run on physical device (camera required)

## Model
Pre-trained MobileNetV2 quantized model from TensorFlow
